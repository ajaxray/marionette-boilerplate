What is this boilerplate?

This boilerplate is a ready setup for starting your next awesome javascript app in minutes.
It includes -

  • Backbone.js
  • MarionetteJS
  • RequireJS
  • Twitter Bootstrap

Credits

I've started with the backbone-marionette-require example from TodoMVC and changed few things to use pages and removed todo related things. Then added Twitter Bootstrap as my personal choice of frontend framework.
